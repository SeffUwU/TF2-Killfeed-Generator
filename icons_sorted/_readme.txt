If you want to add a new icon, you'll need to place .png here.
Rules for name:
1. No quotes. (",',`)
2. Not Special Symbols.

How to add to render list:
1. Go to root/res/
2. open icon.json
3. Scroll down, and add your new icon with the following syntax:
	"<last number + 1>":{"<filename with extention>":"<tags>"}
	Rules:
	1. last number shouldn't be the same as any other before it.
	2. Allowed tags for sorting:

	scout, soldier, pyro, demo,
	heavy, engi, medic, sniper, spy,
	primary, secondary, melee, other

	tags should be separated with a space.
